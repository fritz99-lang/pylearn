 # Security Policy

  ## Supported Versions

  PyLearn follows semantic versioning. Security updates are provided for the latest version.

  | Version | Supported          |
  | ------- | ------------------ |
  | 1.x     | :white_check_mark: |
  | < 1.0   | :x:                |

  ## Reporting a Vulnerability

  If you discover a security vulnerability in PyLearn, please **do not open a public GitHub issue**.

  Instead, email the details to: fritzrobert99@gmail.com with:
  - Description of the vulnerability
  - Steps to reproduce it (if possible)
  - Potential impact

  I'll acknowledge your report within 48 hours and work on a fix. Thank you for helping keep PyLearn secure.
